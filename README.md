# Investigaci-n-y-desarrollo-de-algoritmos-para-el-Problema-de-la-Mochila
Entorno experimental para el Problema de la Mochila. Implementaciones en Julia (secuencial/paralelo), C (benchmarks) y AMPL (modelos exactos). Análisis de eficiencia y exactitud en optimización combinatoria mediante algoritmos de fuerza bruta, dinámica y voraces con apoyo de Python
